# JS_2.0

***Build setUP***

### install dependencies
`npm install`

### serve with hot reload at localhost:8080 (front only)
`npm run dev`

### build for production with minification (front only)
`npm run build`

### build for usage with server on localhost:8080
`webpack`

### launch server 
`node server.js`
